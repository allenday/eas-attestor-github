🔗 EAS Attestor for Github

Link your GitHub username to your Ethereum address with cryptographic proof

1Connect Wallet

Connect your Ethereum wallet to get started. We support MetaMask and other Web3 wallets.

2Sign Message

Enter your GitHub username and sign a verification message.

3Create Gist & Validate

Create a GitHub gist with your signed data and trigger validation.

4Submit to Blockchain

Submit your validated attestation to the blockchain.

🔒 All cryptographic operations happen in your browser. No private keys are transmitted.
